# Patient Visit & Follow-Up Management System

A healthcare-based web application developed using PHP, MySQL, Bootstrap, and Chart.js to efficiently manage patient records, visit details, follow-ups, and healthcare reports.

---

## Features

- Patient CRUD Operations
- Visit & Follow-Up Management
- Search & Filter Functionality
- Pagination for Patient Records
- Session-Based Login & Logout
- Role-Based Access
- Monthly Visits Chart using Chart.js
- Responsive Bootstrap UI
- SQL Joins & Aggregations

---

## Technologies Used

- PHP
- MySQL
- Bootstrap 5
- Chart.js
- HTML5
- CSS3
- JavaScript

---

## Modules

### Patient Module
- Add Patient

- Delete Patient
- View Patient Details

### Visit Module
- Add Visit Records
- Follow-Up Tracking
- Visit History Management

### Reports & Analytics
- Monthly Visits Chart
- Patient Statistics
- Search & Filters
